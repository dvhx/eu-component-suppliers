# eu-component-suppliers
Curated list of EU-based electronic component suppliers

Every shop on this list is physically located within the European Union,
meaning you avoid the new €3 per item customs tariff on non-EU imports.

In the CSV file, the "ships_to" column lists ISO country codes of EU countries 
the supplier ships to, "eu" means all EU member states, "world" means worldwide shipping.

The list is in this CSV file [suppliers.csv](suppliers.csv)
